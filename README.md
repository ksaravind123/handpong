# Hand-Controlled Pong Game (Python)

A webcam-controlled Pong-like game using Python, OpenCV, and MediaPipe/CV-based tracking.

## Features
- Control paddle using your hand (index finger)
- Pinch gesture to launch the ball
- Smooth hand tracking
- Real-time camera preview embedded in-game
